# Traffic-Light-Project
This project uses LEDs to simulate a traffic light.
If you press the "request east traffic" button, it will write west yellow and green low, and write red HIGH, then the east side LEDs will light up from red, yellow, to green.
Vice versa with the "request west traffic" button.
The pedestrian button will set all traffic lights to red, and then light up a white LED simulating a pedestrian light at a crosswalk.
